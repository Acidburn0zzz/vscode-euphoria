# vscode-euphoria

Euphoria language support and debugging for Visual Studio Code

## Features

### Declarative language features

- [x] Syntax highlighting
- [ ] Snippet completion
- [x] Bracket matching
- [x] Bracket autoclosing
- [x] Bracket autosurrounding
- [x] Comment toggling
- [ ] Auto indentation
- [ ] Folding

### Programmatic language features

- [ ] Hover information
- [ ] Auto completion
- [ ] Jump to definition
- [ ] Error checking
- [ ] Formatting
- [ ] Refactoring
- [ ] Folding
